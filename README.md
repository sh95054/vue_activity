# h5.zhuishushenqi.com_activities

Empty project.

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build [activityName]
```

To create a development build:

```sh
npm run watch [activityName]
```

## Running

```sh
node dist/bundle.js
```
